## General Info
This is a browser based web application to recommend users randomized food combos based on their budget, saving them 
time and money. 
	
## Technologies
Technologies used for this project:
* HTML, CSS
* JavaScript
* Jquery
* Bootstrap 
* Firebase Authentication
* Firebase Storage
